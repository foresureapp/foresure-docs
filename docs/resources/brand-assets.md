## Color Palette

Our design system uses a high-contrast theme optimized for readability and "dark mode" aesthetics.

| Color | Hex | Usage |
| :--- | :--- | :--- |
| **Primary Orange** | `#FFA101` | Buttons, Highlights, Accents |
| **Background Dark** | `#0F172A` | Main Background |
| **Surface** | `#1E293B` | Cards, Modals |
| **Text Primary** | `#F8FAFC` | Headings, Body Text |
| **Text Muted** | `#94A3B8` | Secondary Text |

## Typography

We use **Genos** for headings and **Inter** (or system sans-serif) for body text.

*   **Headings**: `Genos`, 700 weight.
*   **Body**: `Inter`, 400 weight.
*   **Monospace**: `Azeret Mono`, for code blocks.
