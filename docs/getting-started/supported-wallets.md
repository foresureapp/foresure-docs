## How to Connect

### Coinbase Wallet (Smart Wallet)
1.  Download the Coinbase Wallet app or extension.
2.  On Foresure, click **Connect**.
3.  Select **Coinbase Wallet**.
4.  Scan the QR code or approve the connection.

### MetaMask
1.  Install MetaMask from [metamask.io](https://metamask.io/).
2.  Ensure you have the **Base Network** added (see [Network Setup](./network-setup.md)).
3.  Click **Connect** -> **MetaMask**.
4.  Approve the connection request.

::: tip Smart Wallet Support
Foresure fully supports **Account Abstraction (ERC-4337)** via Coinbase Smart Wallet, allowing for gasless transactions (sponsored by us for new users!) and passkey logins.
:::
